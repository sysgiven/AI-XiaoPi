# Firmware 目录

本目录用于存放 ESP32 固件相关的文档和链接。

## ESP32 固件

本项目基于 xiaozhi-esp32 硬件，需要相应的 ESP32 固件支持。

### 固件链接
- [xiaozhi-esp32 固件仓库](https://github.com/78/xiaozhi-esp32)

### 烧录说明
1. 下载对应的固件二进制文件
2. 使用 esptool.py 或其他烧录工具将固件烧录到 ESP32
3. 根据硬件配置调整参数

### 硬件配置
- 支持的开发板：ESP32-WROOM、ESP32-WROVER 等
- 需要的传感器和外设：麦克风、扬声器、LED 等
